# Тесты для РК 2

2 варианта запуска программы:
1 - ./app.exe input_file list
2 - ./app.exe input_file output_file form group

## Входные данные
Имя входного файла, в котором хранятся сведения о студентах: фамилия, год поступления, группа
Для первого варианта указывается действие - list
Для второго варианта указывается имя выходного файла, действие form и искомая группа

## Выходные данные
1 - Сформированный список, выведенный на экран
2 - Новый список, записанный файл

## Позитивные тесты:
- 01 - 1 элемент
- 02 - много элементов
2 вариант:
- 02 - 1 студент из искомой группы
- 03 - несколько студентов из искомой группы
- 04 - все студенты из искомой группы

## Негативные тесты:
- 01 - несуществующий файл
- 02 - пустой файл
- 03 - количество ключей меньше 3
- 04 - количество ключей больше 5
- 05 - некорректное действие-аргумент
- 06 - год <= 0
- 07 - вместо фамилии пустая строка
- 08 - вместо группы пустая строка
- 09 - длина фамилии больше 30 символов
- 10 - длина группы больше 10 символов
2 вариант:
- 07 - нет студентов из искомой группы
