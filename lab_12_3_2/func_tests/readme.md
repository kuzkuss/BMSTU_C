# Тесты для 2 задания лабораторной работы №12

## Входные данные
Количество элементов - целое число, выбор способа выделения памяти - целое число 1 или 2, значение для вставки после четных

## Выходные данные
Заполненный простыми числами массив, отфильтрованный массив, в котором после каждого четного добавлено введённое значение

## Позитивные тесты:
- 01 - 1 число в массиве, выделение памяти способом 1
- 02 - 1 число в массиве, выделение памяти способом 2
- 03 - много чисел в массиве, выделение памяти способом 1
- 04 - много чисел в массиве, выделение памяти способом 2

## Негативные тесты:
- 01 - количество элементов <= 0
- 02 - выбор способа выделения памяти не 1 и не 2
