# Тесты для 2 задания лабораторной работы №10

## Входные данные
Действие: out/sps/pos/cat
Для out и sps:
Одна строка
Для pos и cat:
Две строки

## Выходные данные
out:
действие - out
исходная строка
sps:
действие - sps
введённая строка без множественных пробелов
pos:
действие - pos
индекс вхождения подстроки в строку
cat:
действие - cat
результат конкатенации двух строк

## Позитивные тесты:
sps:
- 01 - в строке нет пробелов
- 02 - в строке нет двойных пробелов, но единичные есть
- 03 - в строке в одном месте много подряд идущих пробелов
- 04 - много пробелов в начале строки
- 05 - много пробелов в конце строки
- 06 - все пробелы двойные
- 07 - пустая исходная строка
pos:
- 08 - пустая исходная строка
- 09 - пустая подстрока
- 10 - подстроки нет в исходной строке
- 11 - подстрока в начале исходной строки
- 12 - подстрока в конце исходной строки
- 13 - много подстрок в исходной строке
- 14 - в строке есть и часть подстроки и сама подстрока
cat:
- 15 - первая строка пустая
- 16 - вторая строка пустая
- 17 - длина первой строки делится на размер статического массива (то есть все массивы полностью заполнены)
- 18 - длина первой строки не делится на размер статического массива (то есть последний массив не полностью заполнен)
