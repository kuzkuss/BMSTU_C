# Тесты для 1 задания лабораторной работы №10

Условие:
Во входном файле находятся сведения о необходимых продуктах: название, количество, цена за 1 штуку. Необходимо
при наличии продукта porridge добавить перед ним продукт milk. Если продукт porridge не найден, список оставить неизменным.
В зависимости от последнего параметра командной строки необходимо отсортировать полученный список по убыванию или
возрастанию цены продукта. Результат записать в выходной файл.
Имя входного и выходного файлов указывается в параметрах командной строки. Тип сортировки (по возрастанию/по убыванию)
передаётся последним аргументом командной строки: либо ascending, либо descending.

## Входные данные
Список параметров: ./app.exe INPUT_FILE OUTPUT_FILE ASCENDING
или
./app.exe INPUT_FILE OUTPUT_FILE DESCENDING
В INPUT_FILE находится список структур (название продукта, количество, цена за штуку)

## Выходные данные
ascending:
список структур, отсортированный по возрастанию, при наличии в исходном списке продукта porridge будет
добавлен перед ним продукт milk
descending:
список структур, отсортированный по убыванию, при наличии в исходном списке продукта porridge будет
добавлен перед ним продукт milk

## Позитивные тесты:
ascending:
- 01 - элемент porridge в списке на первом месте
- 02 - элемент porridge в середине списка
- 03 - элемент porridge в конце списка
- 04 - нет элемента porridge в списке
- 05 - один элемент в спике porridge
- 06 - один элемент в спике не porridge
descending:
- 07 - элемент porridge в списке на первом месте
- 08 - элемент porridge в середине списка
- 09 - элемент porridge в конце списка
- 10 - нет элемента porridge в списке
- 11 - один элемент в спике porridge
- 12 - один элемент в спике не porridge

## Негативные тесты:
- 01 - входной файл не существует
- 02 - входной файл пустой
- 03 - количество аргументов командной строки не равно 4
- 04 - вместо названия продукта пустая строка
- 05 - количество продукта меньше или равно 0
- 06 - цена продукта меньше или равна 0
- 07 - вместо количества продукта написаны буквы
- 08 - вместо цены продукта написаны буквы
